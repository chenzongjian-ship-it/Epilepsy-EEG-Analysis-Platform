# 项目说明：癫痫脑电分析平台

## 项目定位

本项目是一个面向课程汇报和作品展示的 EEG 分析平台原型，重点展示癫痫脑电信号分析的完整流程和可视化结果。

## 平台目标

- 用一个 Web 页面串联脑电分析项目的关键步骤。
- 让查看者直观看到 EDF 文件上传后的分析链路。
- 展示波形、时频图、特征表、模型输出和评估指标。
- 提供可通过 GitHub Pages 直接访问的项目入口。

## 核心流程

```text
EDF 文件上传 -> 数据读取 -> 预处理 -> 特征提取 -> 模型训练 -> 结果评估 -> 报告导出
```

## 当前实现

当前版本是纯前端静态原型，适合直接上传到 GitHub 仓库并通过 GitHub Pages 展示。

已实现内容：

- EDF 文件上传入口。
- 分析链路进度展示。
- EEG 原始/滤波波形模拟图。
- 时频展示区域。
- 特征表展示。
- 模型输出和发作概率展示。
- F1-score 和 Sensitivity 展示。
- JSON 分析报告导出。

## 后续真实分析接入方案

后续可增加 Python 后端，将前端上传的 EDF 文件传给分析服务：

1. 使用 MNE 或 PyEDFlib 读取 EDF。
2. 对信号进行滤波、重采样和标准化。
3. 按时间窗切分发作/非发作样本。
4. 提取时域、频域、时频域和非线性特征。
5. 使用 SVM、Random Forest、KNN 或 XGBoost 训练分类模型。
6. 输出混淆矩阵、ROC、F1-score、Sensitivity 等结果。
7. 自动生成项目报告。
