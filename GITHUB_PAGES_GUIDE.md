# GitHub Pages 使用说明

## 上传文件

将以下文件上传到你的 GitHub 仓库根目录：

```text
index.html
styles.css
script.js
README.md
PROJECT_OVERVIEW.md
GITHUB_PAGES_GUIDE.md
.nojekyll
```

## 开启 Pages

在仓库页面中进入：

```text
Settings -> Pages
```

选择：

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

保存后等待 GitHub 构建完成。

## 访问链接

如果你的 GitHub 用户名是 `your-name`，仓库名是 `eeg-analysis-platform`，访问地址通常是：

```text
https://your-name.github.io/eeg-analysis-platform/
```

把这个链接放到 README 的“直接进入平台”位置，就能在仓库首页直接点击进入平台。
